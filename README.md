In modern educational and corporate environments,
efficient and reliable attendance management is crucial. The
Smart Attendance- Based Authentication System leverages
biometric authentication and IoT technology to streamline
the attendance process, ensuring accuracy and security. This
system utilizes an ESP32-WROOM DA module as the core
microcontroller and an R307 fingerprint sensor for biometric
verification. Attendance data is securely recorded and
managed using GOOGLE sheets, providing an accessible and
organized database for administrators.
